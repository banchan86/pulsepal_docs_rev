SendCustomPulseTrain(CustomTrainId, PulseOnset[])

Sends a sequence of onset times and voltages describing a train of pulses.

Remarks

Pulses that are continuous or overlapping will merge. If an output channel is set to produce biphasic pulses, the voltage specified for each pulse is sign-inverted for the second phase, i.e. if +5 V is used for phase 1, -5 V is used automatically for phase 2.

 CustomTrainId

The identity of the custom pulse train to program.

PulseOnset[]

The array specifying all pulse onset times and voltages, respectively in seconds and volts.

 ArgumentException

The specified pulse train id is invalid.

ArgumentNullException

pulseTrain is null.

ArgumentOutOfRangeException

The maximum length of 1,000 pulses has been exceeded.

SendCustomPulseTrain(CustomTrainId, double[], double[])

Sends a sequence of onset times and voltages describing a train of pulses.

Remarks

Pulses that are continuous or overlapping will merge. If an output channel is set to produce biphasic pulses, the voltage specified for each pulse is sign-inverted for the second phase, i.e. if +5 V is used for phase 1, -5 V is used automatically for phase 2.

 CustomTrainId

The identity of the custom pulse train to program.

double[]

The array of pulse onset times, where each time is specified in seconds from the start of the pulse train.

double[]

The array of pulse voltages, with one voltage per pulse.

 ArgumentException

The specified pulse train id is invalid.

ArgumentNullException

Either pulseTimes or pulseVoltages is null.

ArgumentOutOfRangeException

The maximum length of 1,000 pulses has been exceeded.
SendCustomWaveform(CustomTrainId, double, double[])

Sends a sequence of voltages describing a train of continuous monophasic pulses, with periodic onset times.

 CustomTrainId

The identity of the custom pulse train to program.

double

The width of all pulses in the train in the range [0.0001, 3600] seconds. Pulses are continuous.

double[]

The array of pulse voltages, with one voltage per pulse.

 ArgumentOutOfRangeException

samplingPeriod is outside the range [0.0001, 3600] seconds.-or- The maximum length of 1,000 pulses has been exceeded.

ArgumentNullException

pulseVoltages is null.
