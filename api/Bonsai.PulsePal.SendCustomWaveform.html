SendCustomWaveform

sink Operator

Represents an operator that uploads each array of voltages in a sequence to the Pulse Pal device, where each array specifies a custom train of continuous monophasic pulses, with periodic onset times.

Inputs & Outputs

Uploads each vector of voltages in an observable sequence to the Pulse Pal device, where each vector specifies a custom train of continuous monophasic pulses, with periodic onset times.

Mat

A sequence of custom pulse trains, where each pulse train is specified by a row or column vector of pulse voltages. The vector of pulse onset times is automatically computed from the specified sample rate.
Mat

An observable sequence that is identical to the source sequence but where there is an additional side effect of uploading each custom pulse train in the sequence to the Pulse Pal device.
